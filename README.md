## What is Torch?

**Torch** is a curated collection of from-scratch deep learning programs built for people who want to *actually understand* what's happening inside the black box ‚ÄĒ not just import a library and call `.fit()`.

Every program here is intentionally minimal, heavily commented, and designed to isolate one idea at a time. You read it, you run it, you tweak it. That's the loop.

> *"You don't understand it until you can train it from scratch."*

## Program Design Philosophy

Every program in Torch follows these rules:

- **One concept per file** ‚ÄĒ no mixing ideas
- **Comments explain the *why*, not the what** ‚ÄĒ the code shows what; the comments teach
- **Runnable in under 2 minutes** ‚ÄĒ no waiting on giant datasets
- **Breaks are encouraged** ‚ÄĒ modify hyperparameters and watch things fail
